## Usage:
    givemejson [options] <notjson> [yayjson]

## Options:
```
-h, --help                    output usage information
-V, --version                 output the version number
-s, --separator <sep>         Specify the separator used in the input (default is tab)
-S, --output-separator <sep>  Specify the separator use in the output (default is tab)
-l, --line <line>             Specify the line end used (default is \n)
-N, --no-column-names         The data will be treated as if the first line were not the column titles
-T, --dry-run                 Just show the output filename and options
-c, --columns [title]         The column titles to include in the output
-r, --range <a>..<b>          The range (inclusive, row indices start at 1) of row indices to include in the output
```
